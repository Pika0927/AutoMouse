========================================================
Version - 2022/4/15

一般點左上右下，點可由滑鼠移動到指定位置按下F8生成
x1 y1 x2 y2 time(s)
0 0 100 100 5

需要AutoReload則在後方加入"A"
x1 y1 x2 y2 time(s) A
0 0 100 100 5 A

========================================================
Version - 2022/4/22

指令如下 : 
P + FA		//x1 y1 x2 y2 time(s) FA
PIMG + FA	//Image pretime(s) time(s) FA
PIMG2		//Image1 Image2 time(s)
PIMGP		//Image x1 y1 x2 y2 time(s)
CHECK		//Image
END
END BREAK
END STOP
KEY			//paste time(s)
